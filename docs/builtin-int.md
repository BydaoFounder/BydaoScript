### range() - диапазон чисел

Генерирует последовательность целых чисел от начального значения до конечного значения, но не включая его.

**Формат вызова:**

`range( from: Int, to: Int ): IntRange`

**Аргументы:**

`from `- начальное значение диапазона чисел
`to   `- конечное значение диапазона

**Возвращает:**

объект типа `IntRange` или `null`, если аргументы принимают неверные значения, например, в случае `to` < `from`.

**Пример использования:**

```bs
var sum = 0
enum Int.range(0,10) as i { // i принимает значения от 0 до 9 включительно
	sum += i
}
```

### parse() - преобразует строку в число

Выполняет разбор строки и преобразует строчное представление целого числа в целое число.

**Формат вызова:**

`parse( string: Str ): Int`

**Аргументы:**

`string` - строка текста, содержащая представление целого числа

**Возвращает:**

целое число или `null`, если строка не содержит представление целого числа.
