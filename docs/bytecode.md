VarDecl - объявление переменной
    arg1 - индекс строки - названия переменной
    arg2 - определяет способ инициализации:
        0 - без инициализации
        1 - значение со стека
        <0 - значением константы. Для индекса константы нужно поменять знак arg2

VarAdd - сложение переменной с другим значением
    arg1 - индекс переменной левого операнда
    arg2 - индекс правого операнда:
        >0 - индекс переменной
        <0 - индекс константы. Для индекса константы нужно поменять знак arg2
        =0 - значение на стеке

AddStore - прибавить значение к значению переменной
    arg1 - индекс переменной
    arg2 - индекс прибавляемого значения:
        >0 - индекс другой переменной
        <0 - индекс константы (нужно поменять знак)
        =0 - значение на стеке

Store - сохранить значение в переменной
    arg1 - индекс переменной
    arg2 - индекс сохраняемого значения:
        >0 - индекс другой переменной
        <0 - индекс константы (нужно поменять знак)
        =0 - значение на стеке
