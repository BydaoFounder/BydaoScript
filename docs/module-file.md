# File - обработка файлов

Стандартный модуль **File** содержит два набора функций для обработки файлов:
- Функции модуля (в дальнейшем - *функции типа*). Эти функции позволяют выполнять
некоторые операции с файлами без создания объектов файлов;
- функции объектов типа `File`. Эти функции применимы только к объектам.

Для использования функций обработки файлов, необходимо в начале модуля подключить
модуль `File`:
```
use File
```

Создание объекта типа `File` выполняется использованием имени модуля как функции
с аргументом типа строка, который содержит название файла. Название файла может
содержать полный абсолютный путь, относительный путь или не содержать пути.
В качестве разделителя каталогов используется символ '/' для любой операционной
системы. Например:
:
```
var file = File('../docs/manual.md')
```

### File.exists() - проверить существование файла

Проверяет существование заданного файла.

**Формат вызова:**

`File.exists( fileName: Str ): Bool`

**Аргументы:**

- `fileName` - название файла, которое может содержать абсолютный или относительный путь

**Возвращает:**

Возвращает объект типа `Bool`:
- `true` - если файл существует на диске;
- `false` - если файл не существует или не найден.

**Пример использования:**

```bs
if File.exists( 'test-file.test' ) {
    // выполнить необходимые операции с файлом
    // ...
}
```

### File.copy() - копирование файла

Копирует файл в другое место и/или под другим именем.

> [!NOTE]
> Перед копированием рекомендуется проверять сущестование исходного файла.

**Формат вызова:**

`File.copy( fileNameFrom: Str, fileNameTo: Str ): Bool`

**Аргументы:**

- `fileNameFrom` - название исходного файла, которое может содержать абсолютный или относительный путь
- `fileNameTo`   - целевое название файла, которое может содержать абсолютный или относительный путь

**Возвращает:**

Возвращает объект типа `Bool`:
- `true` - если копирование было успешным;
- `false` - если копирование не удалось.

**Пример использования:**

```bs
if File.exists( 'test-file.test' ) {
    var ok = File.copy( 'test-file.test', 'test-file.copy' )
    // обработать результат копирования
    // ...
}
```

### File.move() - перемещение файла

Перемещает файл файл в другое место.

> [!NOTE]
> Перед перемещением рекомендуется проверять сущестование исходного файла.

**Формат вызова:**

`File.move( fileName: Str, target: Str ): Bool`

**Аргументы:**

- `fileName` - название исходного файла, которое может содержать абсолютный или относительный путь
- `target`   - целевое название файла или путь, куда нужно переместить файл. Целевой путь
 должен существать перед перемещением.

**Возвращает:**

Возвращает объект типа `Bool`:
- `true` - если перемещение было успешным;
- `false` - если перемещение не удалось.

**Пример использования:**

```bs
if File.exists( 'test-file.test' ) {
    var ok = File.move( 'test-file.test', './archiv' )
    // обработать результат копирования
    // ...
}
```

**Формат вызова:**

- `newFileName` - новое название файла без пути.

**Возвращает:**

Возвращает объект типа `Bool`:
- `true` - если переименование было успешным;
- `false` - если переименование не удалось.

**Пример использования:**

```bs
var ok = File.rename( 'path/to/file/test-file.test', 'test.file' )
if ! ok {
    // обработать ошибку переименования
    // ...
}
```

### File.readAll() - чтение из файла

Читает все содержимое файла в строку.

**Формат вызова:**

`File.readAll( fileName: Str ): Str`

**Аргументы:**

- `fileName` - название файла, которое может содержать абсолютный или относительный путь

**Возвращает:**

В случае ошибки чтения файла возвращает `null`. В случае успешного чтения файла,
возвращает объект типа `Str` - строку, содержащую всё содержимое файла.

**Пример использования:**

```bs
if File.exists( 'test-file.test' ) {
    var content = File.readAll( 'test-file.test' )
    if content == null {
        // обработать ошибку чтения
        // ...
    }
}
```

**Формат вызова:**

`File.writeAll( fileName: Str, data: Str ): Bool`

**Аргументы:**

- `data`     - строка, всё содержимое которой будет записано в файл

**Возвращает:**

Возвращает объект типа `Bool`:

**Пример использования:**

```bs
if File.exists( 'test-file.test' ) {

    }
}
```


### open() - открыть файл

Открывает файл в заданном режиме для операций чтения и/или записи.

**Формат вызова:**

`open( mode: Str ): Bool`

**Аргументы:**

- `mode`- режим открытия файла. Строка, которая может содержать комбинацию флагов:
  - 'r' - открыть файл для чтения
  - 'w' - открыть файл для записи и очистить его содержимое, если файл уже существует
  - 'a' - открыть файл для записи в конец файла (для добавления в файл)
  - '+' - открыть файл для чтения и записи
  - 't' - открыть файл в текстовом режиме (как текст)
  - 'b' - открыть файл в двоичном режиме

**Возвращает:**

объект типа `Bool`:
- `true` - если файл успешно открыт в заданном режиме;
- `false` - если не удалось открыть файл

**Пример использования:**

```bs
var file = File('name.txt')
if ! file.open('rt') {
    Sys.outln( 'Cannot open file ' + file.name )
}
```


### close() - закрыть файл

Закрывает файл, делая его недоступным для операций чтения и/или записи.

**Формат вызова:**

`close(): Void`

**Аргументы:**

нет аргументов

**Возвращает:**

ничего не возвращает

**Пример использования:**

```bs
var file = File('name.txt')
if ! file.open('rt') {
    Sys.outln( 'Cannot open file ' + file.name )
}
file.close()
```

### read() - прочитать из файла

Читает из файла все данные или заданное количество символов.

**Формат вызова:**

`read( count: Int = -1 ): Str`

**Аргументы:**

- `count` - число символов для чтения. По умолчанию - все символы.

**Возвращает:**

объект типа `Str` - строку с прочитанными данными. Если файл не открыт, возвращает
значение `null`.

**Пример использования:**

```bs
var file = File('name.txt')
if ! file.open('rt') {
    Sys.outln( 'Cannot open file ' + file.name )
}
var content = file.read()   // прочитать все содержимое файла
if content == null {
    Sys.outln( 'Cannot read file ' + file.name )
}
file.close()
```

### readln() - прочитать строку из файла

Читает из файла одну строку текста.

**Формат вызова:**

`readln(): Str`

**Аргументы:**

нет

**Возвращает:**

объект типа `Str` - строку с прочитанными данными. Если файл не открыт или из файла
прочитаны все строки, возвращает значение `null`.

**Пример использования:**

```bs
var file = File('name.txt')
if ! file.open('rt') {
    Sys.outln( 'Cannot open file ' + file.name )
}
var line = file.readln()   // прочитать первую строку текста
while line != null {
    // обработать текущую строку line
    // ...
    line = file.readln()   // прочитать следующую строку текста
}
file.close()

// Вариант обработки строк файла в стиле BydaoScript:

var line = file.readln()   // прочитать первую строку текста
while line != null next line = file.readln() {
    // обработать текущую строку line
    // ...
}
```


### readlnAll() - прочитать все строки из файла

Читает из файла все строки текста.

**Формат вызова:**

`readlnAll(): Array`

**Аргументы:**

нет

**Возвращает:**

объект типа `Array` - массив, в котором каждый элемент - это строка из файла.
Если файл не открыт, возвращает значение `null`. Массив может быть пустым.

**Пример использования:**

```bs
var file = File('name.txt')
if ! file.open('rt') {

file.close()
```


### write() - записать в файл

Записывает в файл заданные данные.

**Формат вызова:**

`write( data: Str ): Bool`

**Аргументы:**

- `data` - строка с данными для записи

**Возвращает:**

Если файл не открыт, возвращает значение `null`. Иначе, возвращает объект типа `Bool`:
- `true` - если запись выполнена успешно;
- `false` - если не удалось записать данные в файл.

**Пример использования:**

```bs
var file = File('name.txt')
if ! file.open('wt') {
    Sys.outln( 'Cannot open file ' + file.name )
}
var ok = file.write( 'Новое содержимое файла' )
if ! ok {
    Sys.outln( 'Cannot write to file ' + file.name )
}
file.close()
```

### append() - добавить в файл

Если файл был открыт, запоминает режим открытия файла и закрывает файл.
Открывает файл в режиме добавления ("at") и добавляет в конец файла переданную строку.
Закрывает файл и, если файл был открыт ранее, открывает его в первоначальном режиме.

**Формат вызова:**

`append( data: Str ): Bool`

**Аргументы:**

- `data` - строка с данными для записи

**Возвращает:**

Возвращает объект типа `Bool` с результатом добавления в файл:
- `true` - если запись выполнена успешно;
- `false` - если не удалось записать данные в файл.

**Пример использования:**

```bs
var file = File('name.txt')
var ok = file.append( 'Дополнительные данные' )
if ! ok {
    Sys.outln( 'Cannot append to file ' + file.name )
}
```


### pos() - получить текущую позицию

Получить текущую позицию в файле после открытия или последней операции чтения/записи.

**Формат вызова:**

`pos( ): Int`

**Аргументы:**

нет

**Возвращает:**

Если файл не открыт, возвращает значение `null`. Иначе, возвращает объект
типа `Int` - текущую позицию в файле

**Пример использования:**

```bs
var file = File('name.txt')
if ! file.open('wt') {
    Sys.outln( 'Cannot open file ' + file.name )
}
var currentPos = file.pos()
Sys.outln( 'Current position: ' + currentPos )  // Вывод: Current position: 0
file.close()
```


### seek() - установить текущую позицию

Устанавливает новую текущую позицию в файле, с которой будет выполняться операция
чтения или записи.

**Формат вызова:**

`seek( pos: Int ): Bool`

**Аргументы:**

- `pos` - заданная новая позиция

**Возвращает:**

Если файл не открыт, возвращает значение `null`. Иначе, возвращает объект типа `Bool`:
- true - если позиция успешно установлена;
- false - если не удалось установить позицию.

**Пример использования:**

```bs
var file = File('name.txt')
if ! file.open('rt') {
    Sys.outln( 'Cannot open file ' + file.name )
}
if ! file.seek(10) {
    Sys.outln( 'Cannot seek to position 10' )
}
var content = file.read()
Sys.outln( 'Content from position 10: ' + content )
file.close()
```


### atEnd() - проверить конец файла

`atEnd(): Bool`

**Возвращает:**

типа `Bool`:
- `true` - если текущая позиция указателя находится в конце файла;

**Аргументы:**

нет
